## 🎯 Deployment Overview

### **Backend (Railway)**

**What:** .NET 10.0 API with SQLite database  
**Where:** Railway.app (free tier)  
**How:** Push to GitHub → Deploy on Railway  
**Time:** 15 minutes  
**Result:** Backend URL (e.g., `https://your-app.up.railway.app`)

**Steps:**
1. Create GitHub repository
2. Upload backend files
3. Deploy to Railway
4. Get backend URL
5. Test Swagger documentation

### **Frontend (Evo Builder)**

**What:** React app with TypeScript  
**Where:** Evo Builder hosting  
**How:** Set environment variable → Publish  
**Time:** 10 minutes  
**Result:** Public URL (e.g., `https://your-app.evo.builder`)

**Steps:**
1. Set `VITE_API_URL` environment variable
2. Test locally
3. Click "Publish" button
4. Get public URL
5. Test published app

### **Marketing**

**What:** QR code, email, posters  
**Where:** Company-wide distribution  
**How:** Use templates provided  
**Time:** 15 minutes  
**Result:** Company-wide awareness

**Steps:**
1. Generate QR code
2. Send email (template provided)
3. Print posters (templates provided)
4. Display in common areas

## 🔑 Important Information

### **URLs You'll Need**

1. **Backend URL** (from Railway)
   - Format: `https://your-app-name.up.railway.app`
   - Use in: `VITE_API_URL` environment variable
   - Test at: `https://your-app-name.up.railway.app/swagger`

2. **Frontend URL** (from Evo Builder)
   - Format: `https://your-app.evo.builder` (or custom domain)
   - Use in: Emails, posters, QR codes
   - Share with: Your entire company

### **Admin Access**

- **Method:** Click soccer ball ⚽ in bottom-right corner
- **Password:** `Fryingpan.420!` (case-sensitive)
- **Features:**
  - View all entries
  - See statistics (total entries, goals predictions, etc.)
  - Export CSV
  - View player selection counts

### **Tournament Details**

- **Date:** March 20, 2026
- **Location:** Sydney
- **Teams:** Men's Team (10 players) + Mixed Team (10 players)
- **Games:** 3-6 per team (30 minutes each)
- **Prize:** Premier League shirt (winner's choice)
- **Donation:** $10 minimum to Heartbeat of Football
- **Deadline:** March 19, 2026 (day before tournament)

### **Tie-Breaker Rules**

1. **First:** Closest total goals prediction
2. **If tied:** Combined goals by predicted top scorers (highest wins)
3. **If still tied:** Earliest submission time
