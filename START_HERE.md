## 🐛 Troubleshooting

### "Backend not found"
→ Set `VITE_API_URL` environment variable in Evo Builder

### "CORS error"
→ Backend has CORS configured, check Railway logs

### "Admin won't open"
→ Triple-tap quickly (within 1 second), password is case-sensitive

### "Can't submit entry"
→ Check all fields filled, image uploaded, backend running

**More help**: [DEPLOYMENT_GUIDE.md](DEPLOYMENT_GUIDE.md) (Troubleshooting section)
